# 🏘️ שכונה מרקט — MVP

שוק שכונתי קהילתי. מחליף קבוצות ווטסאפ עמוסות בלוח מובנה לפריטים חינם, מכירה ובקשות.

---

## 🛠️ Stack טכנולוגי

| שכבה | כלי | סיבה |
|------|-----|-------|
| Frontend | Next.js 14 App Router + TypeScript | SSR, routing, PWA-ready |
| עיצוב | Tailwind CSS | פיתוח מהיר, RTL מלא |
| Backend | Supabase (Postgres + Auth + Storage + Realtime) | הכל-ב-אחד, ללא שרת נפרד |
| Auth | Supabase Phone OTP | ישראלי, ללא סיסמאות |
| תמונות | Supabase Storage | משולב, ללא עלות נוספת |
| Realtime | Supabase Realtime | צ'אט חי ללא תשתית |
| Deploy | Vercel | חינמי, חיבור אוטומטי ל-GitHub |

---

## 📋 דרישות מוקדמות

- Node.js 18+
- חשבון [Supabase](https://supabase.com) (חינמי)
- חשבון [Vercel](https://vercel.com) (חינמי)
